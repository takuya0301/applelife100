About

「林檎生活100」は、僕が Web からこれまでの人生で享受した利益を Web に還元するために始めました。

「林檎生活100」では、Blog という形式で日々の実験を記事として書いています。この記事が検索によって発見され、発見したその人の利益になれば、ひとつの還元になるのではないでしょうか。

これからの人生でも Web から利益を享受しつづけるでしょうが、還元もしつづけられればと思っています。
